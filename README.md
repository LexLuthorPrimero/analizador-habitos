# 🧠 Analizador de Hábitos Diario

Proyecto básico en Python que permite registrar hábitos diarios y calcular métricas de productividad.

---

## 🚀 Funcionalidades

- Registro de múltiples hábitos
- Cálculo de tiempo total
- Promedio de tiempo por actividad
- Resumen ordenado en consola

---

## 🛠️ Tecnologías

- Python 3
- Lógica básica (listas/diccionarios)
- Funciones modulares

---

## ▶️ Cómo ejecutar

```bash
python main.py
